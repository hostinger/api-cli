## hostinger vps firewall deactivate

Deactivate firewall

### Synopsis

Deactivate a firewall for a specified virtual machine.

Use this endpoint to remove firewall protection from VPS instances.

```
hostinger vps firewall deactivate <firewall-id> <virtual-machine-id> [flags]
```
