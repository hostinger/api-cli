## hostinger agency-hosting domains link-to-plan-website

Link domain to Agency Plan website

### Synopsis

Links a domain to the specified Agency Plan website so it can serve traffic for that domain.

```
hostinger agency-hosting domains link-to-plan-website <website_uid> [flags]
```

### Options

```
      --domain string   Fully qualified domain name to link to the website
  -h, --help            help for link-to-plan-website
```
